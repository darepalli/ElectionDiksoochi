Methodology & Sources

India Election Compass Transparency Page

Data and Privacy Notice

We do not collect, store, or sell personal user data from compass responses. Compass answers are processed in-browser to calculate match scores.

How Theses Are Built

Theses are drafted from publicly available party manifestos, official speeches, policy documents, government publications, court/commission records, and mainstream public reporting.

Each party-position entry in the election config supports a dedicated source URL. As source links are completed, they appear next to position explanations in the compare view.

Core Public Resources

Party and Candidate Sources

Use official party websites, manifesto PDFs, press releases, and verified social media handles as primary references. Prefer direct documents over commentary whenever possible.

Corrections and Disputes

If a party or citizen identifies an incorrect position, submit a correction with a documentary source link. We review evidence and update entries transparently.

This project is non-partisan and for civic information only. It does not endorse any party or candidate.

Browser and Mobile Compatibility

The compass is designed and tested for modern evergreen browsers on desktop and mobile.

  • Desktop: Latest 2 versions of Chrome, Edge, Firefox, and Safari
  • Mobile: Latest 2 versions of Android Chrome and iOS Safari

Known limitations:

  • Internet Explorer is not supported
  • Very old browsers may fail because JavaScript APIs are required
  • JavaScript must be enabled for compass calculations and source popups
  • Some compare-screen layouts can feel dense on very small mobile screens in portrait mode
  • External source links depend on the availability of third-party websites
